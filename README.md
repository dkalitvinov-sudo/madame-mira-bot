# Madame Mira Telegram Bot ✨

Telegram-бот для продажи разборов ($11 / $29)

## Возможности

- Ответы на сообщения
- Воронка продаж
- Подготовка к интеграции GPT
- Готов к деплою на Render

## Запуск

1. Установить зависимости:
pip install -r requirements.txt

2. Запустить:
python app.py

## Переменные окружения

TELEGRAM_TOKEN = ваш токен бота
